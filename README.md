# [Character Optimizer](https://marcelomiyazaki.github.io/RPG_CharacterOptimizer/)

A web application made for mobile services with the goal to help RPG's players to manage their health and mana points, and in addiction organize their skill tests and damage rolls.

## Functions:

1. Health and Mana points manager with automatic save
2. Proper calculator to add and subtract HP and MP
4. Retractable notepad
5. Regular dice rolling simulator `[d4, d6, d8, d10, d12, d20]`
6. Custom roll creator with automatic save `[you can choose what dice and how many you wanna roll, with a modifier or not]`

To add a new custom roll, you need to press the `+` button, and to wipe all the created rolls, you need to press the `reset` button

The name area is only to identify the weapon, magic, skill... you are using

The d4, d6, d8, d10, d12 and d20 spaces correspond the number of that type of dice you will roll when using this custom roll

The mod box is the number that will be added to the final result of the dice, that's the modifier

To use this custom roll, you need to click the `roll` button in the side of the name, the result will be showed in the result area above the `+` and `reset` buttons
